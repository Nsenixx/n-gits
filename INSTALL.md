Dependencies (Server)
------------

* python 3.x >= 3.4


External python packages
------------

* PyYAML
* tornado

		$ pip3 install requirements.txt


Dependencies (Client)
------------
* nvm
* npm

		$ nvm install 6
		$ nvm use 6
		$ npm install

Try finally this
------------

**Run tests**
	
		$ python3 gits/test/terminal_test.py

**Run client**:
		
		$ npm start

**Run server** (start server on default 8888 port):
		
		$ python3 server.py

Go to your lovely browser (url - localhost:8888)
